# 📘 Learning Log – Java Project

A simple terminal-based Java application to add and view learning topics.  

## 🚀 Features

- ➕ Add a new learning topic  
- 📋 View all topics with ID and created timestamp  
- 🚫 Prevent duplicate topic names  
- ❌ Prevent empty topic names  
- 💻 Simple terminal-based interface  

## 🛠️ Technologies Used

- Java
- Intellij

## 📌 Example Menu

```
1. Add Topic
2. View Topics
3. Exit
```
